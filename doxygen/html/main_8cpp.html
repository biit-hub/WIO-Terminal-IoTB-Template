◆ periodicTasks()

void periodicTasks ( )

In dieser Funktion werden Aufgaben und Funktionen, nach Ablauf eines bestimmten Intervalls, ausgeführt.

Zu beachten
Für eigene periodische Funktionen muss das Array previousMillis erweitert werden.
previousMillis[3] und userDefinedIntervall können für eine eigene periodische Aufgabe benutzt werden.
